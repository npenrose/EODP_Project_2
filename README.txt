READMEDescriptions---------------This repository contains five python files which create a combination of csv files and png files.To run our code and replicate our results, simply go to each of the files and run them individually.# PYTHON FILES #preprocessing.py: Contains all of the preprocessing steps, including normalization and discretizationstatistical_analysis.py: Calculates the relationship between number of ratings and average rating for books and the variation in ratings of the top books by the age groupTop_rated_author.py: Finds the top rated author for the target age groupCorr_analysis.py: Finds the cosine similarity between author ratings for users between the ages of 23-33User_Based_Collaborative_Filtering.py: The implementation of the recommendation system
